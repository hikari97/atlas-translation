# Architecture Decision Records (ADR)

## Purpose

Architecture Decision Records (ADRs) capture significant architectural decisions made throughout the Atlas Studio project.

Each ADR documents:

- Context
- Decision
- Consequences
- Alternatives considered

ADRs provide historical context and prevent the same discussions from recurring.

---

# Lifecycle

```text
Proposed
    ↓
Accepted
    ↓
Superseded
    ↓
Deprecated
```

---

# Naming

```
ADR-0001-use-monorepo.md

ADR-0002-package-dependency-rules.md

ADR-0003-document-object-model.md
```

---

# Template

Every ADR follows the same structure.

- Status
- Context
- Decision
- Consequences
- Alternatives
- References
