# Glossary

## Application

Public runtime façade.

---

## Runtime Kernel

Internal runtime engine.

---

## Bootstrap

Startup orchestration pipeline.

---

## Module

Primary runtime composition unit.

---

## Registry

Stores definitions.

---

## Provider

Registers runtime capabilities.

---

## Factory

Creates runtime instances.

---

## Resource

Any object with lifecycle management.

---

## Runtime Context

Shared runtime execution context.

---

## Environment

Execution platform abstraction.

---

## Configuration

Immutable runtime configuration.

---

## Service

Reusable runtime capability.
