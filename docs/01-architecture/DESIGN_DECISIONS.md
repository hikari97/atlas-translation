# Design Decisions

## DD-0001

Application is immutable.

---

## DD-0002

Bootstrap uses Pipeline pattern.

---

## DD-0003

Runtime is coordinated through Runtime Kernel.

---

## DD-0004

Modules are the composition root.

---

## DD-0005

Service registration is separated from service creation.

---

## DD-0006

Configuration is immutable after bootstrap.

---

## DD-0007

Runtime components communicate through contracts.

---

## DD-0008

Internal implementation is hidden behind public APIs.
