# TASK-0024 — Create Executable Usage Examples

## Summary

Create executable usage examples for the Atlas Studio Runtime Framework.

Examples serve as the official learning resource for framework users while also acting as integration tests to verify that the public API remains functional.

Every example must compile and execute successfully.

---

# Goal

Provide production-quality examples that demonstrate the intended usage of the Atlas Studio Runtime Framework.

Examples should remain synchronized with the public API through automated validation.

---

# Background

Documentation becomes outdated quickly when examples are treated as static text.

Atlas Studio adopts executable documentation.

Every published example must:

- Compile successfully.
- Execute successfully.
- Reflect current APIs.
- Be validated by Continuous Integration.

