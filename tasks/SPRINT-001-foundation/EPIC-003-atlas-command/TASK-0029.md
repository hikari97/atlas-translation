# Hardening Checklist

## 1. Build Integrity

Verify:

- Production build succeeds
- Declaration files generated
- Source maps generated (if enabled)
- Build output deterministic

---

## 2. API Integrity

Verify:

- Public API frozen
- No accidental exports
- No internal exports
- Tree shaking preserved
- Declaration files complete

---

## 3. Dependency Integrity

Verify:

- ADR-0002 satisfied
- No circular dependencies
- No forbidden imports
- Dependency graph valid
- No duplicate runtime dependencies

---

## 4. Security

Verify:

- No vulnerable dependencies
- No hardcoded secrets
- No debug code
- No development artifacts
- Safe default configuration

---

## 5. Performance

Verify:

- Benchmark baseline maintained
- Performance budget satisfied
- No unexpected allocations
- No obvious regression

---

## 6. Release Integrity

Verify:

- Package metadata correct
- Version correct
- License present
- Changelog updated
- Repository metadata valid
- Publish configuration correct
