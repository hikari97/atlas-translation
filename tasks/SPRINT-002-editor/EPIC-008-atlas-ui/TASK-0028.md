# TASK-0028 — Implement Resize Handle

## Summary

Implement the Resize Handle component for Atlas UI.

The Resize Handle provides a reusable interaction component that enables users to resize editor objects such as images, tables, videos, embeds, shapes, and plugin-defined blocks. It integrates with the Editor View, Interaction Layer, Theme System, Accessibility Layer, and Resize infrastructure while remaining framework-independent.

The Resize Handle is responsible only for initiating and visualizing resize interactions. It must not resize content or mutate document state.

---

# Goal

Provide a reusable, accessible, and extensible Resize Handle component capable of supporting object resizing through a stable public API.

---

# Background

Resizable objects are common throughout modern editors.

Typical use cases include:

- Image resizing
- Table resizing
- Video resizing
- Embed resizing
- Shape resizing
- Plugin-defined resizable objects

Atlas UI should expose a single Resize Handle abstraction reusable across all editor objects.

# Responsibilities

The Resize Handle is responsible for:

- Displaying resize controls.
- Initiating resize interactions.
- Tracking resize state.
- Integrating with Theme System.
- Providing accessibility metadata.

The Resize Handle is **not** responsible for:

- Resizing objects.
- Updating layouts.
- Editing documents.
- History management.
- Business logic.
