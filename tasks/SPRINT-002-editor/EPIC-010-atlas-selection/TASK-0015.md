# TASK-0015 — Implement Selection Range Utilities

## Summary

Implement the `SelectionRangeUtilities`, a reusable collection of utility functions responsible for performing common range selection operations within Atlas Studio.

The Selection Range Utilities provide deterministic algorithms for calculating, comparing, normalizing, intersecting, merging, and querying selection ranges. These utilities are shared across range selection, marquee selection, keyboard navigation, hierarchical selection, and plugin-defined selection workflows.

The Selection Range Utilities are responsible only for performing pure range-related computations. They do not maintain selection state, render visual feedback, execute commands, or modify application state.

---

# Goal

Provide a reusable, extensible, and framework-independent utility library capable of supporting every range-based selection workflow through a stable public API.

---

# Background

Many selection features require identical range operations.

Typical examples include:

- Comparing two ranges
- Normalizing start/end positions
- Merging overlapping ranges
- Splitting ranges
- Detecting intersections
- Calculating range length
- Plugin-defined range utilities

Rather than duplicating these algorithms throughout Atlas Studio, they should exist within a centralized utility module.

# Responsibilities

The Selection Range Utilities are responsible for:

- Normalizing ranges.
- Comparing ranges.
- Merging compatible ranges.
- Detecting intersections.
- Producing immutable calculation results.
- Supporting future utility extensions.

The Selection Range Utilities are **not** responsible for:

- Managing selections.
- Rendering selection overlays.
- Updating documents.
- Executing commands.
- Business logic.

# Public API

Example:

```ts
interface SelectionRangeUtilities {
  normalize(range: SelectionRange): SelectionRange;

  merge(first: SelectionRange, second: SelectionRange): SelectionRange;

  intersect(
    first: SelectionRange,
    second: SelectionRange,
  ): SelectionRange | null;

  contains(range: SelectionRange, item: SelectionItem): boolean;
}
```

Example range:

```ts
interface SelectionRange {
  readonly start: number;

  readonly end: number;
}
```
