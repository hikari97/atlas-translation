# TASK-0007 — Implement Keyboard Event

## Summary

Implement the `KeyboardEvent`, the immutable abstraction responsible for representing keyboard interactions throughout Atlas Studio.

The Keyboard Event abstraction provides a normalized, platform-independent representation of keyboard input generated by physical keyboards, virtual keyboards, accessibility devices, and future input providers. It standardizes key information, modifier states, repeat behavior, timing metadata, and event lifecycle while remaining completely independent from browser APIs, operating system event models, rendering engines, application frameworks, and editor implementations.

The Keyboard Event abstraction is responsible only for representing keyboard events. It does not execute shortcuts, modify keyboard state, dispatch events, or implement business logic.

---

# Goal

Provide a reusable, immutable, and framework-independent keyboard event abstraction capable of representing every keyboard interaction through a stable public API.

---

# Background

Operating systems and browsers expose different keyboard event models.

Typical differences include:

- Key identifiers
- Scan codes
- Modifier handling
- Repeat behavior
- Dead keys
- IME interactions
- Virtual keyboards
- Accessibility devices

Rather than exposing platform-specific events, Atlas Studio should expose one normalized Keyboard Event abstraction.

# Responsibilities

The Keyboard Event abstraction is responsible for:

- Representing immutable keyboard events.
- Preserving normalized key information.
- Providing modifier metadata.
- Recording repeat information.
- Supporting future keyboard providers.

The Keyboard Event abstraction is **not** responsible for:

- Dispatching events.
- Updating keyboard state.
- Executing shortcuts.
- Rendering.
- Business logic.

# Public API

Example:

```ts
interface KeyboardEvent {
  readonly id: string;

  readonly type: KeyboardEventType;

  readonly key: string;

  readonly code: string;

  readonly repeat: boolean;

  readonly timestamp: number;

  readonly modifiers: ModifierState;

  readonly metadata: KeyboardEventMetadata;
}
```

Example event type:

```ts
type KeyboardEventType = 'keydown' | 'keyup' | 'keypress' | 'repeat';
```
