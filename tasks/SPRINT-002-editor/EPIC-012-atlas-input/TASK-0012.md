# TASK-0012 — Implement Pointer Event

## Summary

Implement the `PointerEvent`, the immutable abstraction responsible for representing all pointer interactions within Atlas Studio.

The Pointer Event abstraction provides a normalized, platform-independent representation of pointer input generated by mice, touchscreens, styluses, trackpads, pen tablets, accessibility devices, and future pointer providers. It standardizes pointer identity, position, movement, pressure, tilt, button states, timing metadata, and event lifecycle while remaining completely independent from browser APIs, operating system event models, rendering engines, application frameworks, and editor implementations.

The Pointer Event abstraction is responsible only for representing pointer events. It does not dispatch events, manage pointer state, recognize gestures, render interfaces, or implement business logic.

---

# Goal

Provide a reusable, immutable, and framework-independent pointer event abstraction capable of representing every pointer interaction through a stable public API.

---

# Background

Modern operating systems expose different pointer event models.

Typical examples include:

- Mouse movement
- Mouse button events
- Touch contacts
- Stylus movement
- Pen pressure
- Tilt information
- Hover detection
- Accessibility pointing devices

Rather than exposing browser-specific or platform-specific pointer events, Atlas Studio should expose one normalized Pointer Event abstraction.

# Responsibilities

The Pointer Event abstraction is responsible for:

- Representing immutable pointer events.
- Preserving normalized pointer information.
- Providing device-specific metadata.
- Recording timing information.
- Supporting future pointer providers.

The Pointer Event abstraction is **not** responsible for:

- Dispatching events.
- Updating pointer state.
- Recognizing gestures.
- Rendering.
- Business logic.

# Public API

Example:

```ts
interface PointerEvent {
  readonly id: string;

  readonly type: PointerEventType;

  readonly pointerId: number;

  readonly pointerType: 'mouse' | 'touch' | 'pen' | 'custom';

  readonly button: number;

  readonly buttons: number;

  readonly position: Point2D;

  readonly pressure: number;

  readonly tiltX: number;

  readonly tiltY: number;

  readonly twist: number;

  readonly timestamp: number;

  readonly metadata: PointerEventMetadata;
}
```

Example event type:

```ts
type PointerEventType =
  | 'pointerdown'
  | 'pointermove'
  | 'pointerup'
  | 'pointercancel'
  | 'pointerenter'
  | 'pointerleave'
  | 'pointerover'
  | 'pointerout';
```
