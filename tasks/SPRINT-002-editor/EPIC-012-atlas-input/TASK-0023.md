# TASK-0023 — Implement Shortcut Resolver

## Summary

Implement the `ShortcutResolver`, the abstraction responsible for deterministically resolving keyboard shortcuts within Atlas Studio.

The Shortcut Resolver evaluates normalized keyboard input against registered shortcut definitions while considering active shortcut contexts, keyboard layouts, modifier states, key mappings, shortcut priorities, and user-defined overrides. It guarantees deterministic shortcut selection regardless of operating system, browser, application framework, or input source.

The Shortcut Resolver is responsible only for matching shortcut definitions. It does not execute commands, modify application state, render interfaces, or implement business logic.

---

# Goal

Provide a reusable, deterministic, and framework-independent shortcut resolution abstraction capable of selecting the correct shortcut definition through a stable public API.

---

# Background

Professional applications frequently register hundreds or even thousands of shortcuts.

Examples include:

- Global shortcuts
- Editor shortcuts
- Canvas shortcuts
- Layer shortcuts
- Modal shortcuts
- Plugin shortcuts
- User-defined shortcuts
- Extension-defined shortcuts

Multiple shortcut definitions may match the same keyboard input.

Example:

```text
Ctrl+C

↓

Global

↓

Editor

↓

Plugin

↓

Modal Dialog
```

The Shortcut Resolver determines which shortcut should be activated according to deterministic resolution rules.

# Resolution Rules

Resolution should evaluate:

- Active shortcut context
- Parent contexts
- Shortcut priority
- Modifier state
- Keyboard layout
- Key mapping
- Shortcut enabled state
- User overrides

Resolution order must remain deterministic.
