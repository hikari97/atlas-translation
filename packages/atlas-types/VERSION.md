# Version

Current version: `0.1.0`

## Stability

This is the initial foundation release for internal Atlas Studio integration.

## Compatibility

The package targets TypeScript strict consumers and Node.js `>=18.17.0` for package tooling.

## Versioning Policy

- Patch releases: documentation and non-breaking declaration fixes.
- Minor releases: additive public types, enum members, or grouped exports.
- Major releases: breaking contract changes or removed public exports.
