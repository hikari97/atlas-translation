# Domains

## Foundation

Common primitives, branded IDs, JSON-safe values, result types, geometry, and shared enums.

## Workspace

Workspace, project, and page contracts used by document and application packages.

## Editor

Editor state, layer, bubble, typography, font, selection, and canvas contracts.

## Translation

Translation content, provider, confidence, review, tag, history, glossary, and reference contracts.

## Resource

Asset and image contracts for files, metadata, locations, checksums, thumbnails, and image properties.

## Platform

Plugin, workflow, history, and export contracts for extensibility and platform-level operations.
