# Atlas Validation API

The public API is exported from `src/index.ts`.

Primary contracts:

- `ValidationCore`
- `ValidationContract`
- `ValidationContext`
- `ValidationRegistry`
- `ValidationResolver`
- `ValidationRule`
- `ValidationExecutor`
- `ValidationMetadata`
- `ValidationLifecycle`
- `ValidationBuilder`
- `ValidationProvider`
- `ValidationEngine`
- `ValidationFactory`

Factory helpers create provider-neutral registries, contexts, validation contracts, rules, resolvers, executors, providers, engines, and factories.
