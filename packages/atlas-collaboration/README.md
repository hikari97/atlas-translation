# @atlas/atlas-collaboration

Provider-independent collaboration contracts for Atlas Studio.

The package defines collaboration sessions, presence, awareness, shared documents, synchronization snapshots, conflict resolution, remote cursors, remote selections, comments, suggestions, reviews, annotations, shared clipboard, follow-user state, permissions, events, and history.

It does not implement networking, WebSocket, WebRTC, CRDT, Operational Transform, storage, rendering, or UI.
