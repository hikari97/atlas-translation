# Release Notes

## 0.1.0

Initial `atlas-document` release.

### Added

- Atlas document aggregate root.
- Workspace, project, page, layer, bubble, and translation document classes.
- Generic document collection framework.
- Serialization, traversal, snapshot, diff, and mutation frameworks.
- Root and subpath public API exports.
- Documentation, type tests, and release readiness reports.

### Verification

- `npm run typecheck`
- `npm run build`
- `npm run test:types`
- `npm pack --dry-run`
