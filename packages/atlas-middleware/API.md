# Atlas Middleware API

The public API is exported from `src/index.ts`.

Primary contracts:

- `MiddlewareCore`
- `MiddlewareContract`
- `MiddlewareContext`
- `MiddlewarePipeline`
- `MiddlewareChain`
- `MiddlewareRegistry`
- `MiddlewareResolver`
- `MiddlewareExecutor`
- `MiddlewareMetadata`
- `MiddlewareLifecycle`
- `MiddlewareBuilder`
- `MiddlewareProvider`
- `MiddlewareEngine`

Factory helpers create provider-neutral registries, contexts, pipelines, middleware contracts, resolvers, executors, providers, and engines.
