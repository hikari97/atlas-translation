# Atlas Controller API

The public API is exported from `src/index.ts`.

Primary contracts:

- `ControllerCore`
- `ControllerContract`
- `ControllerContext`
- `ControllerRegistry`
- `ControllerResolver`
- `ControllerDispatcher`
- `ControllerExecutor`
- `ControllerMetadata`
- `ControllerLifecycle`
- `ControllerBuilder`
- `ControllerProvider`
- `ControllerEngine`
- `ControllerFactory`

Factory helpers create provider-neutral registries, contexts, controller contracts, resolvers, dispatchers, executors, providers, engines, and factories.
